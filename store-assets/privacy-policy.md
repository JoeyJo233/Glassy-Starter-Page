# GlassyStart Privacy Policy / 隐私政策

Last updated: 2026-04-11

## English

### Overview

GlassyStart is a browser new tab extension that provides a customizable start page, including search, bookmarks, folders, wallpapers, and appearance settings.

We aim to minimize data collection. Most user data is stored locally in the user's browser.

### What Data GlassyStart Uses

GlassyStart may process the following data as part of its core functionality:

- Search queries entered by the user
- Bookmark and folder data created by the user
- Appearance and layout settings chosen by the user
- Wallpaper selections and user-uploaded wallpaper files
- Search history stored locally for convenience

### How Data Is Stored

GlassyStart stores most extension data locally on the user's device.

- Settings and bookmark data may be stored in local browser storage
- User-uploaded wallpaper data may be stored in IndexedDB on the local device
- Search history may be stored locally in the browser

GlassyStart does not require users to create an account.

### Third-Party Requests

To provide search and search suggestion features, GlassyStart may send user-entered search text to the selected third-party search provider, such as:

- Google
- Bing
- DuckDuckGo
- Yandex

These requests are only made when the user uses the search function or search suggestions. Those third-party services process requests under their own privacy policies and terms.

GlassyStart may also load remote images or icons when the user uses online wallpapers, website favicons, or external search engine icons.

### Data Sharing

GlassyStart does not sell user data.

GlassyStart does not share personal data with data brokers.

GlassyStart does not use user data for advertising, credit evaluation, or unrelated profiling.

### Data Collection by the Developer

The developer of GlassyStart does not operate a backend account system or remote database for this extension's core features.

The developer does not intentionally collect or store users' bookmark content, wallpaper files, or local settings on developer-controlled servers.

### Permissions

GlassyStart may request browser permissions needed for its core functionality, including storage-related capabilities and access to search suggestion endpoints used by selected search engines.

These permissions are used only to support extension features.

### Your Choices

Users can control their data by:

- Editing or deleting bookmarks and folders
- Clearing local browser storage
- Removing uploaded wallpapers
- Disabling or uninstalling the extension

### Contact

If you have questions about this privacy policy, you can contact the publisher through the contact information provided on the Chrome Web Store listing.

## 中文

### 概述

GlassyStart 是一个浏览器新标签页扩展，提供可自定义的起始页功能，包括搜索、书签、文件夹、壁纸和外观设置。

我们尽量减少数据收集。大多数用户数据仅存储在用户本地浏览器中。

### GlassyStart 会处理哪些数据

为了实现核心功能，GlassyStart 可能会处理以下数据：

- 用户输入的搜索词
- 用户创建的书签和文件夹数据
- 用户选择的外观与布局设置
- 壁纸选择以及用户上传的壁纸文件
- 为了方便使用而保存在本地的搜索历史

### 数据如何存储

GlassyStart 的大部分数据都存储在用户设备本地。

- 设置和书签数据可能存储在浏览器本地存储中
- 用户上传的壁纸数据可能存储在本地设备的 IndexedDB 中
- 搜索历史可能存储在浏览器本地

GlassyStart 不要求用户注册账号。

### 第三方请求

为了提供搜索和搜索联想功能，GlassyStart 可能会将用户输入的搜索文本发送给用户所选择的第三方搜索服务提供商，例如：

- Google
- Bing
- DuckDuckGo
- Yandex

这些请求仅在用户主动使用搜索功能或搜索联想功能时发生。相关第三方服务会按照其自身的隐私政策和服务条款处理这些请求。

当用户使用在线壁纸、网站 favicon 或外部搜索引擎图标时，GlassyStart 也可能加载远程图片或图标资源。

### 数据共享

GlassyStart 不会出售用户数据。

GlassyStart 不会向数据经纪商共享个人数据。

GlassyStart 不会将用户数据用于广告投放、信用评估或与扩展核心功能无关的用户画像。

### 开发者是否收集数据

GlassyStart 的开发者没有为该扩展核心功能运营账号系统或远程数据库。

开发者不会主动在自己控制的服务器上收集或存储用户的书签内容、壁纸文件或本地设置。

### 权限说明

GlassyStart 可能会请求浏览器权限，以支持扩展核心功能，包括与本地存储相关的能力，以及访问所选搜索引擎搜索联想接口所需的网络权限。

这些权限仅用于支持扩展功能本身。

### 用户可做的控制

用户可以通过以下方式管理自己的数据：

- 编辑或删除书签和文件夹
- 清除浏览器本地存储
- 删除已上传的壁纸
- 禁用或卸载扩展

### 联系方式

如果你对本隐私政策有疑问，可以通过 Chrome Web Store 商店页面中提供的联系方式联系发布者。
