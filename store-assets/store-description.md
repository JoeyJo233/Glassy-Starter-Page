# GlassyStart Store Description / 商店描述

## English

### Short Description

GlassyStart is a customizable new tab extension with glassmorphism design, smart search, bookmarks, folders, and wallpaper personalization.

### Detailed Description

GlassyStart transforms your browser's new tab page into a clean, modern, and highly customizable start page.

Built with a glassmorphism-inspired visual style, GlassyStart helps you access search, bookmarks, folders, and personalized backgrounds in one focused workspace.

#### Key Features

- Beautiful glassmorphism new tab design
- Multiple search engine support
- Search suggestions and local search history
- Drag-and-drop bookmark grid
- Folder support for organizing shortcuts
- Custom icons and favicon support
- Wallpaper presets and user-uploaded wallpapers
- Adjustable blur, opacity, scale, spacing, and text colors
- Import and export of bookmarks and settings

#### Search Experience

GlassyStart supports multiple search engines, allowing you to switch between providers and quickly search from your new tab page. Search suggestions help surface relevant queries while local history makes repeated searches faster.

#### Personalization

Users can customize wallpapers, transparency, blur, text color, layout offsets, and overall scale to create a start page that fits their own workflow and aesthetic preferences.

#### Bookmark Organization

Create a shortcut dashboard with drag-and-drop reordering, folders, and custom icons, making frequently used websites easier to access.

#### Privacy

Most settings and user data are stored locally in the browser. Search suggestions are requested from the selected search provider only when the user enters search text.

## 中文

### 简短描述

GlassyStart 是一个可自定义的新标签页扩展，提供毛玻璃风格界面、智能搜索、书签管理、文件夹和壁纸个性化功能。

### 详细描述

GlassyStart 将浏览器的新标签页改造成一个干净、现代、且高度可自定义的起始页。

它采用毛玻璃风格设计，让你可以在一个更聚焦的界面中集中使用搜索、书签、文件夹和个性化背景。

#### 核心功能

- 美观的毛玻璃新标签页设计
- 支持多个搜索引擎切换
- 搜索联想与本地搜索历史
- 支持拖拽排序的书签网格
- 支持文件夹组织常用网站
- 支持自定义图标和网站 favicon
- 支持预设壁纸和用户上传壁纸
- 支持调节模糊、透明度、缩放、间距和文字颜色
- 支持书签和设置的导入导出

#### 搜索体验

GlassyStart 支持多个搜索引擎，用户可以从新标签页快速切换搜索服务并直接发起搜索。搜索联想可以帮助你更快找到目标内容，本地历史则有助于重复搜索时提升效率。

#### 个性化能力

用户可以自定义壁纸、透明度、模糊程度、文字颜色、布局偏移和整体缩放，打造更符合自己工作流和审美习惯的新标签页。

#### 书签管理

通过拖拽排序、文件夹整理和自定义图标，GlassyStart 可以帮助你构建一个更高效的快捷入口面板，方便访问常用网站。

#### 隐私说明

大多数设置和用户数据仅保存在本地浏览器中。搜索联想仅在用户输入搜索内容时，请求用户所选择的搜索服务提供商。
